# Projeto de Interface
Nossa principal preocupação ao projetar a interface é torná-la intuitiva e eficiente, além de ser adequadamente otimizada para dispositivos móveis. Dessa forma, buscamos garantir que o programa mantenha sua identidade visual em todas as páginas.
